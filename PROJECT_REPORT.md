# Autonomous AI Financial Research Agent

## Objective

To build an AI-powered financial research agent capable of collecting financial data, company news, SEC information, earnings data, and generating a structured financial research report.

## Technologies

* Python
* yFinance
* ChromaDB
* Requests

## Modules

* Planner
* Financial Tool
* News Tool
* SEC Tool
* Earnings Tool
* Memory
* Conflict Resolution
* Report Generator

## Result

The system successfully performs autonomous financial research and generates a structured report.

## Conclusion

The project demonstrates a modular AI-agent architecture with automated financial analysis and report generation capabilities.
